# MT4_Public
FX自動取引MT4開発
MT4というFX機械取引ソフト上で動く、取引実行ファイルの開発のための関連のソースコードをアップロードします。
実行ファイルはEA(expert adviser)といい、そのソースコードはmq4ファイルです。言語はマイナーで、MQL4と言います。

もし動かしたい方は、MT4をインストールし、ある程度ネット上や書物での説明を読み所定のフォルダに、ソースコードを入れてコンパイルしてください。

アップしたソースコードのうち、
#include <MyLib.mqh>
が書かれているものは、MyLib.mq4およびMyLib.mqhを使用しないと動きません。
